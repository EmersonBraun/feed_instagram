## Geting starter

### Run backend: 

```bash
$  git clone https://github.com/EmersonBraun/feed_instagram.git
$ cd feed_instagram/backend
$ yarn 
$ yarn dev 
```

### Run Frontend: 

```bash
$ cd frontend 
$ yarn 
$ yarn start
```

### Run Mobile:

```bash
$ cd mobile
$ yarn 
# Android 
$ react-native run-android
#iOS 
$ react-native run-ios
```
